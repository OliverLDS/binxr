# Cancel all Binance Options orders by underlying

Cancel all Binance Options orders by underlying

## Usage

``` r
options_cancel_all_orders_by_underlying(underlying, config = config_options())
```

## Arguments

- underlying:

  Underlying symbol, for example `"BTCUSDT"`.

- config:

  An options configuration created by
  [`config_options()`](https://oliverlds.github.io/binxr/reference/config_options.md).

## Value

A parsed list.
