# Get Binance Options server time

Get Binance Options server time

## Usage

``` r
options_get_server_time(config = config_options())
```

## Arguments

- config:

  An options configuration created by
  [`config_options()`](https://oliverlds.github.io/binxr/reference/config_options.md).

## Value

A `POSIXct` timestamp.
