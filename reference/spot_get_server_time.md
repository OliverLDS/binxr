# Get Binance Spot server time

Get Binance Spot server time

## Usage

``` r
spot_get_server_time(config = config_spot())
```

## Arguments

- config:

  A spot configuration created by
  [`config_spot()`](https://oliverlds.github.io/binxr/reference/config_spot.md).

## Value

A `POSIXct` timestamp.
