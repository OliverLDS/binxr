# Get Binance Options underlying index price

Get Binance Options underlying index price

## Usage

``` r
options_get_index_price(underlying, config = config_options())
```

## Arguments

- underlying:

  Underlying spot pair, for example `"BTCUSDT"`.

- config:

  An options configuration created by
  [`config_options()`](https://oliverlds.github.io/binxr/reference/config_options.md).

## Value

A parsed list.
