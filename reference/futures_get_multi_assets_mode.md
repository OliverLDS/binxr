# Get Binance Futures multi-assets mode

Get Binance Futures multi-assets mode

## Usage

``` r
futures_get_multi_assets_mode(config = config_futures())
```

## Arguments

- config:

  A futures configuration created by
  [`config_futures()`](https://oliverlds.github.io/binxr/reference/config_futures.md).

## Value

A parsed list.
